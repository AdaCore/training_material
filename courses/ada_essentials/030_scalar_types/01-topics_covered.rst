================
Topics Covered
================

----------------
Topics Covered
----------------

* Integer-based types

  * Signed integers

    * Types defined by range rather than bits

  * Modular types

    * Unsigned types that support bitwise operations and wrap

* Floating-point types

  * Real numbers that specify relative precision

* Enumeration types

  * Ordered collections of logical values

* Attributes

  * Operations to determine properties of types

* Subtypes

  * Adding constraints to an existing type

* Derived types

  * Creating a new distinct type from an existing type

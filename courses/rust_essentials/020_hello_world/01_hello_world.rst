Hello, World
--------------

.. code:: rust

  // Our first program!
  fn main() {
      println!("Hello World!");
  }

* :rust:`//` - indicates a single-line **comment**
* :rust:`fn` - "introduces" a **function**
* :rust:`{ }` - curly braces enclose a **block**
* :rust:`main` - the entry point of the program
* :rust:`println!` - macro for printing a string followed by newline
